# Zombie_Survive
4방향에서 리스폰 되는 좀비를 제거하는 게임

## 개발
- 엔진 : Unity
- 언어 : C#
- 주요 에셋/패키지 : AI Navigation, Cinemachine, Post Processing, Fun 2

## 조작방법
- 이동 : 방향키 or WASD
- 공격 : 마우스 L클릭
- 재장전 : R키

## 게임 특징
- 4방향에서 3가지 종류의 좀비가 출현한다.
- 좀비는 특성에 따라 기본형, 탱커형, 스피드형이 있다.
- 스테이지가 올라감에 따라 출현 좀비의 수가 증가한다.
- 코인(점수), 하트(체력), 탄환(공격수) 증가해주는 아이템이 출현하다.
